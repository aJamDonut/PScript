So what does it do?

Put simply, YummyBerry combines a powerful backend written in PHP and a resposive jQuery powered UI for the frontend.

 

This gives the developer the freedom to create rich interactive web pages on the fly, without all the hassle of designing or implementing a heavy framework.

How does it work?

YummyBerry provides a small folder structure and PHP class to build your app inside, the frontend transitions and state will be taken care of by the JS engine on the frontend.

Fast

Yummy is built with speed in mind. It won't slow you down by making you learn new ways to do things you already do well.

Yummy empowers you to choose how simple or complex your application will be.

Easy

If you know PHP and can dabble with Javascript you'll love Yummy. It's built with novice developers in mind and simplifies many aspects of setting up a rich internet application.

Free

The source code is totally free and can be editted and redistributed at your own discretion.

SEO Friendly

YummyBerry uses HTML5's pushState() technology to maintain state - this means all code generated by YummyBerry can be crawled and indexed by search engines
